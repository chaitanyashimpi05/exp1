Experiment No. 1
Title:
Introduction to Kali Linux Interface, Basic Commands, File Management, System
Information, Networking, Process Management, and Compression Commands

Objective:
To familiarize students with the Kali Linux interface and practice essential Linux commands
for file management, system info, networking, process handling, and compression.

Software/Tools Required:
 Kali Linux OS (2023 or later)
 Terminal access
 Basic user account

Theory:
Kali Linux Interface Overview:
Kali Linux is a Debian-based Linux distribution used for penetration testing and
cybersecurity. Its interface includes:
 Desktop Environment (Xfce by default)
 Terminal Emulator
 Applications Menu
 Taskbar &amp; Notification Area

1. Basic Linux Commands:
Command Description
pwd Shows current working directory
ls Lists files and directories
cd Changes the directory
clear Clears the terminal
whoami Displays the current user
man &lt;command&gt; Displays manual page of a command

2. File Management Commands:
Command Description
touch file.txt Creates a new empty file
mkdir folder Creates a new directory
cp file1 file2 Copies file1 to file2
mv old new Moves or renames a file/directory
rm file.txt Deletes a file
rmdir dir Deletes an empty directory
cat file.txt Displays contents of the file
nano file.txt Opens file in terminal editor

3. System Information Commands:
Command Description
uname -a Shows kernel and system info
hostname Displays the system’s hostname
uptime Shows how long the system has been up
top Displays live system processes
free -h Displays memory usage
df -h Shows disk usage of all partitions

4. Networking Commands:
Command Description
ifconfig / ip a Shows IP address and interfaces
ping &lt;host&gt; Sends ICMP echo to check network connectivity
netstat -tuln Shows open ports and services
traceroute &lt;host&gt; Traces route to a network host
nslookup &lt;domain&gt; DNS lookup of a domain name

5. Process Management Commands:
Command Description
ps Displays running processes
ps aux Detailed view of all processes
kill &lt;PID&gt; Terminates process with given PID
top Interactive real-time view of processes

htop Enhanced process viewer (if installed)

6. Compression Commands:
Command Description
tar -cvf file.tar folder/ Creates a tar archive
tar -xvf file.tar Extracts tar archive
gzip file.txt Compresses file to file.txt.gz
gunzip file.txt.gz Decompresses file.txt.gz
zip file.zip file.txt Zips the file
unzip file.zip Extracts a zip file
